# QR Coupon Redemption & Wallet System

## Overview
This project is a backend system where users can redeem QR coupons and wallet gets updated.

## Features
- Coupon validation
- Wallet credit system
- Idempotent API handling
- Concurrency handling
- Reconciliation API

## APIs
- POST /redeem
- GET /wallet
- POST /admin/reconcile

## How to Run
1. Clone repo
2. Setup database
3. Run project using Visual Studio

## Tech Stack
- .NET 8
- Entity Framework Core
- SQL Server
